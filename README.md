# Interactive Online Resume

Frontend application for an interactive, multilingual online resume.

The project is designed to present professional experience, projects, and skills in a clean and structured format, with a layout optimized for PDF export.

---

## Features

- Multilingual content support
- Dynamic data loading via API
- Clean and minimal UI optimized for PDF export
- Responsive layout
- Lightweight state management

---

## Tech Stack

- Vue 3
- Pinia
- W3.CSS
- JavaScript (ES6+)

---

## Backend

This project uses a dedicated Go backend service to provide dynamic content via API.

- Backend repository: https://github.com/mtima97/go-sandbox

---

## Project Purpose

This project was created as a personal resume website to demonstrate practical backend and frontend skills, including API design, localization, and clean 
UI implementation suitable for real-world usage.
